Tentang Saya

Halo! Saya Walidhani Faturrohman, lulusan SMK Negeri Situraja jurusan Rekayasa Perangkat Lunak. Saya memiliki minat yang besar di bidang teknologi, khususnya dalam pengembangan web menggunakan HTML, CSS, dan PHP.

Selama mengikuti Praktik Kerja Lapangan (PKL) di Cyber Labs, saya terlibat dalam proyek pembuatan website testimoni makanan, yang membantu saya mengasah kemampuan dalam membuat tampilan web yang menarik dan fungsional.

Saya adalah pribadi yang teliti, semangat belajar tinggi, dan senang mengeksplorasi hal-hal baru di dunia pemrograman. Saya percaya bahwa proses belajar tidak berhenti, dan saya selalu terbuka untuk mengembangkan diri melalui pengalaman baru maupun kerja sama tim.

Proyek

Berikut adalah beberapa proyek yang telah saya kerjakan selama belajar dan praktik di bidang pengembangan web. Proyek-proyek ini mencerminkan keterampilan saya dalam HTML, CSS, dan PHP, serta kemampuan saya dalam menciptakan tampilan web yang menarik dan fungsional:

Website Testimoni Makanan

Proyek ini melibatkan pembuatan website yang menampilkan testimoni dari pengguna tentang berbagai jenis makanan.

Website Kasir

Website kasir sederhana yang memungkinkan pengguna untuk menjelajahi produk, menambahkan dan menghapus produk, serta melakukan pembelian. Proyek ini juga mencakup fitur untuk menghitung total harga pembelian dan memberikan ringkasan transaksi. Website ini dirancang untuk memudahkan pengguna dalam melakukan transaksi dan melacak riwayat pembelian mereka.

Website Portofolio

Portofolio pribadi saya yang menampilkan keterampilan dan pengalaman saya dalam pengembangan web.
